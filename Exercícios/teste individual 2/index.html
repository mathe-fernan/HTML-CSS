Agora vamo para uma lista

As listas são boas porque você pode elencar coisas né, por uma ordenada, tipo essa:

  1. Item um
  2. Item dois
  3. Item três

Podemos ainda ter uma ordenada, mas com outros caracteres de ordenação, tipo essa:

  1. Item um
  2. Item dois
  3. Item três

    4. Essa:

    1. Item um
    2. Item dois
    3. Item três

    E essa:

    1. Item um
    2. Item dois
    3. Item três

    Essas daqui são ordenadas, mas você pode escolher por qual colocação começa:

    1. coloca só esse item
    2. e esse

    Agora temos listas não-ordenadas. Elas não tem ordenação, então é só uma lista tipo de supermercado memo:

    • Item um
    • Item dois
    • Item 3

    Dá pra mudar o ícone de cada item da lista também

    • Só um item

    E dá pra misturar os dois tipos também

    1. Uma coisa
      • coisa dois
